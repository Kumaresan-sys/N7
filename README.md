# N7 Landing Page

Responsive Next.js implementation of the central N7 banking landing page from the provided Figma reference.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- lucide-react icons

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```

## Notes

The Figma MCP design-context tool was unavailable during implementation, so the page is built from the accessible central-frame reference and structured for quick pixel tuning once full Figma exports or MCP access are available.
